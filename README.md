# Ohjelmistotekniikka, harjoitustyö

Kurssin harjoitustyöksi olisi tarkoituksena tehdä ns. "virtuaalinen kirjahylly", luettujen ja vielä lukemattomien kirjojen kirjanpito(:D)sovellus.
Hieman tarkempi alustava kuvaus alla.

Käyttäjä voisi lisätä sovellukseen kirjoja, joita on jo lukenut, lukee tällä hetkellä tai joita haluaisi lukea tulevaisuudessa. Kirjoilla olisi eri ominaisuuksia, kuten nimi, kirjailija, genre, kansikuva jne. Käyttäjä pystyisi myös lajittelemaan kirjoja esim. sen mukaan, minkä kirjan seuraavaksi haluaisi lukea. Lopulliset ominaisuudet varmasti tarkentuvat vielä.
